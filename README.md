# Progetto_Biblioteca
Sistema di gestione biblioteca, permette di gestire varie sottoclassi di risorse.
Sono presenti una classe utente che può prendere in prestito e restituire libri e una classe biblioteca che contiene un elenco di utenti e di risorse.
Essa permette di trovare libri e utenti.
Il programma permette di creare nuovi utenti, libri, ebook e riviste, inserire e visualizzare le informazioni relative a loro e prendere in prestito e restituire una risorsa.

## Istruzioni
Scegliere nel menu quale azioni si vuole eseguire.
Dal menu si può aggiungere un utente,aggiungere una risorsa, stampare l'inventario, prendere in prestito una risorsa,restituire una risorsa o uscire dal programma.
